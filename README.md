# Monte Carlo Algorithms Library

A comprehensive library of Monte Carlo and quasi-Monte Carlo algorithms for numerical computation and statistical analysis.

## Structure

### Non-uniform distributions
Implementations of algorithms for generating random variates from various probability distributions.

### Numerical quadrature using (quasi) Monte Carlo
Monte Carlo and quasi-Monte Carlo methods for numerical integration and quadrature.

### Statistical properties of (quasi) Monte Carlo
Analysis and evaluation of statistical properties of Monte Carlo and quasi-Monte Carlo sequences.

### Variance reduction methods
Variance reduction techniques to improve the efficiency of Monte Carlo simulations.
