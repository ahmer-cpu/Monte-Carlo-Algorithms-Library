# Monte Carlo Algorithms Library

A comprehensive library of Monte Carlo and quasi-Monte Carlo algorithms for numerical computation and statistical analysis.

## Project Website

- Website: https://ahmernadeem.com/research/numerical-monte-carlo-library.html
- This page contains a project overview and detailed programming reports describing the code, tasks, experiments, results (including figures), and conclusions.

## Repository Structure

- `Non-uniform distributions/` - Implementations of algorithms for generating random variates from various probability distributions. **Language:** Julia and Python.
- `Numerical quadrature using (quasi) Monte Carlo/` - Monte Carlo and quasi-Monte Carlo methods for numerical integration and quadrature. **Language:** Jupyter Notebook (Python).
- `Statistical properties of (quasi) Monte Carlo/` - Analysis and evaluation of statistical properties of Monte Carlo and quasi-Monte Carlo sequences. **Language:** MATLAB (`.m`, `.mlx`).
- `Variance reduction methods/` - Variance reduction techniques to improve the efficiency of Monte Carlo simulations. **Language:** Julia and MATLAB (`.m`, `.mlx`).
